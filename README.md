# colors-lamp

A web application built using the LAMP stack. 
The user logs into their account, adds whatever color they desire to their color database, 
and then later onthey are able to search for the color. 

# Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- Apache

# Folder Layout

    index.html

  README.md
  LICENSE.md
  .gitignore

# Setup Instructions
1. Clone this repo
2. Place files in server directory
3. Configure database credentials in Login.PHP
4. Import database schema
5. Navigate to http://localhost/ (this can be your i.p. server or your local one (3000))
